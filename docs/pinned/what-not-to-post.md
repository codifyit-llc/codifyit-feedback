# What Not to Post Here

Do not post the following in this public repo:

- security disclosures
- private client information
- implementation details covered by NDA or contract
- support incidents needing immediate attention
- billing or contract questions
- public development tracking requests

If your message includes confidential or client-specific information, use the
private channel already established with CodifyIT instead of posting here.

If a discussion lands here but does not fit the boundary, a maintainer will
redirect it rather than handling it publicly.
